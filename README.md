

Este projeto consiste em um site desenvolvido para realizar consulta de CEP, busca por logradouro e cálculo de frete de forma simples, rápida e intuitiva.

A aplicação simula funcionalidades presentes em sistemas de e-commerce e plataformas logísticas, permitindo ao usuário obter informações de endereço e estimativa de envio a partir dos dados informados.
